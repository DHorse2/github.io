More... This is a utility created to transform data between various flat and database formats.

 This project included many objectives not strictly related to database import. Data transfer between databases using a standard import/export format is fairly well addressed in most products.

 This product will prove most useful in dealing with formats that do not come directly from a database or require cleaning and transformation.

 The intial targets addressed pulling data from a post-relational database to create an MS SQL or MY SQL schema.
