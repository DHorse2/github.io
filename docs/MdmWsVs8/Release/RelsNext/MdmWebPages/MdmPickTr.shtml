PickTR serial communications application

The PickTR was created in response to a difficult problem in client support. This partly consisted of hardware flaws in communications boards and independent of that, very poor and inconsistent line conditions for communications.

PickTR is a file transfer program similar to many in most respects. This version runs on a multivalued host computer. In basic terms, files are transfered between the hosts using a variation on the Kermit protocol.

However, PickTR is truly unique in other respects. It was not only able to transmit successfully and solve the two driving problems, it is able to transmit under the very worst of line conditions.

PickTR uses a strategy that varies many characterists of both the line settings and protocol. Using this strategy in a highly dynamic fashion not only solved a very difficult problem but represented a significant advance in the communications technology in use.
