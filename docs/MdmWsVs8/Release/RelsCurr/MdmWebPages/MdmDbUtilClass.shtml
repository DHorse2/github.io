The File Subsystem

The file subsystem is a large set of classes intended to handle Ascii, binary, SQL databases and post-relational object databases (also known as mulit-valued or PICK databases)

These classes are not intended to address the rudimentary needs of simple applications. These require little more that a basic connection string and may already have classes to handle security such as user name and password. Permission and authorization may already be in place or built into the application.
